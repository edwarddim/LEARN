# Commands

## Basics
```
git init

git add .

git commit -m "YOUR MESSAGE HERE"

git push

git reset
```

## Branches
```
git checkout -b <feature_branch>

git checkout <main or master branch>

git merge <feature_branch>

git branch -d <feature_branch>

```
